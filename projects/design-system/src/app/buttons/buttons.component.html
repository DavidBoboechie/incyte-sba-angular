Buttons

We use Bootstrap's default classes that we then overwrite to define our own styling to them.

The button offers many personalization options such as its colors and layout.

Simple button

The button just requires the .btn class applied and then can be personalized with the .btn-primary, .btn-secondary and .btn-outline classes.

Using Sinequa's [sq-action-buttons] directive allows you to put the additional class in the style option's parameter.

Below are the examples with the button elements and using the directive (with a forced blue background behind the outline button to see it better, being white).

Floating action button

To setup the FAB mode, you need to add the .fab-btn class to the div.

Dropdown button

The dropdown is to be setup on the Angular action options, no additional styling is necessary for the button.
